📘 Tworzenie i modyfikowanie elementów DOM

W JavaScript można dynamicznie tworzyć, dodawać i usuwać elementy na stronie. Dzięki temu strona może się zmieniać bez przeładowania.

🔹 Najważniejsze metody

  • document.createElement("tag") – tworzy nowy element
  • document.createTextNode("tekst") – tworzy węzeł tekstowy
  • element.appendChild(child) – dodaje element jako ostatnie dziecko
  • element.prepend(child) – dodaje element na początek
  • element.insertBefore(newEl, refEl) – wstawia nowy element przed wskazanym
  • element.remove() – usuwa element z DOM
  • parent.replaceChild(newEl, oldEl) – zamienia element

📖 Dodatkowe metody

  • element.cloneNode(true) — kopiuje element (true = z dziećmi)
  • element.innerHTML = "tekst" — ustawia zawartość HTML wewnątrz elementu
  • element.textContent = "tekst" — ustawia zwykły tekst (bez HTML)
  • element.insertAdjacentHTML("beforeend", "<p>Nowy</p>") — wstawia kod HTML w konkretne miejsce

📊 Podsumowanie

  • createElement() — tworzy nowy element
  • appendChild(), prepend() — dodają element do DOM
  • insertBefore() — wstawia w konkretnym miejscu
  • replaceChild() — zamienia element
  • remove() — usuwa element

Dzięki tym metodom możesz dynamicznie manipulować strukturą strony – dodawać, zamieniać lub usuwać elementy w reakcji na zdarzenia.
