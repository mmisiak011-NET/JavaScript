📘 DOM — metody pobierania elementów

JavaScript pozwala pobierać elementy z dokumentu HTML za pomocą różnych metod. Najpopularniejsze z nich to:

  • document.getElementById("id") — pobiera element po ID
  • document.getElementsByClassName("class") — zwraca kolekcję elementów o podanej klasie
  • document.getElementsByTagName("tag") — zwraca wszystkie elementy danego typu (np. div, p)
  • document.querySelector("selector") — zwraca pierwszy element pasujący do selektora CSS
  • document.querySelectorAll("selector") — zwraca wszystkie dopasowane elementy (NodeList)

🎯 Zasady używania selektorów w querySelector()

Selektory działają dokładnie tak, jak w CSS. Oto podstawowe zasady:

Selektor Opis Przykład
"#id" Wybiera element po jego ID document.querySelector("#header")
".class" Wybiera element po klasie CSS document.querySelector(".message")
"tag" Wybiera element po nazwie znacznika document.querySelector("p")
"div.message" Wybiera <div> z klasą message document.querySelector("div.message")
"section p" Wybiera elementy <p> znajdujące się wewnątrz <section> document.querySelector("section p")

📊 Podsumowanie

  • getElementById() — szybki dostęp do pojedynczego elementu
  • getElementsByClassName() / getElementsByTagName() — zwracają „żywe” kolekcje
  • querySelector() — najbardziej uniwersalna metoda (CSS-selektory)
  • querySelectorAll() — zwraca wszystkie dopasowania jako NodeList
