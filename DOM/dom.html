📘 Co to jest DOM (Document Object Model)

DOM to obiektowa reprezentacja dokumentu HTML, którą przeglądarka tworzy po wczytaniu strony. Przedstawia strukturę strony w postaci drzewa obiektów, które JavaScript może odczytywać i modyfikować.

🧩 Przykład struktury DOM

    Document
    └── html
          ├── head
          │    └── title
          └── body
              ├── h1
              └── p

Każdy element HTML (np. <div>, <p>, <img>) to węzeł DOM (node).

⚙️ DOM a HTML

HTML to tekstowy dokument, a DOM to jego obiektowa wersja w pamięci przeglądarki. Dzięki temu JavaScript może dynamicznie zmieniać zawartość strony bez przeładowania.

🔹 Najczęstsze operacje na DOM

  • document.getElementById("id") — pobiera element po ID
  • document.querySelector("selector") — pobiera pierwszy pasujący element
  • element.textContent — pobiera lub ustawia tekst
  • element.style.color = "red" — zmienia styl elementu
  • document.createElement("p") — tworzy nowy element
  • element.addEventListener("click", fn) — dodaje obsługę zdarzenia

📊 Podsumowanie

  • DOM to model obiektowy strony — „żywa” reprezentacja HTML.

  • Można dynamicznie zmieniać tekst, style, strukturę i reagować na zdarzenia.
