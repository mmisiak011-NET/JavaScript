📘 JavaScript — Najczęściej używane zdarzenia (Events)

Źródła i dokumentacja: MDN Web Docs: Web Events • Window events • W3Schools event list

🪟 Zdarzenia obiektu window

NazwaOpis
loadWszystkie zasoby (HTML, CSS, obrazy) zostały załadowane.
DOMContentLoadedDOM gotowy, ale niekoniecznie obrazy i CSS.
resizeRozmiar okna został zmieniony.
scrollUżytkownik przewija stronę.
focus / blurOkno lub element uzyskał / stracił fokus.
online / offlinePołączenie z internetem przywrócone / utracone.
beforeunloadUżytkownik próbuje opuścić stronę.
hashchangeFragment URL po znaku # został zmieniony.
popstateUżytkownik porusza się po historii (przycisk wstecz/dalej).
storageZmieniono dane w localStorage (w innej karcie).
messageOtrzymano wiadomość z innego okna/ramki (postMessage API).
errorBłąd w skrypcie lub zasobie.

🖱️ Zdarzenia myszy

NazwaOpis
clickKliknięcie myszą.
dblclickPodwójne kliknięcie.
mousedown / mouseupNaciśnięcie / zwolnienie przycisku myszy.
mousemoveRuch kursora myszy.
mouseenter / mouseleaveKursor wchodzi / opuszcza element.
contextmenuKliknięcie prawym przyciskiem (menu kontekstowe).

⌨️ Zdarzenia klawiatury

NazwaOpis
keydownNaciśnięcie klawisza (trzymany — powtarzany).
keyupZwolnienie klawisza.
keypress(Stare) Wciśnięcie klawisza znakowego.

🧾 Formularze i pola

NazwaOpis
inputWartość pola zmieniona (w czasie pisania).
changeWartość zmieniona i pole opuszczone.
focus / blurPole uzyskało / straciło fokus.
submitWysłanie formularza.
resetReset formularza.

🎞️ Multimedia

NazwaOpis
play / pauseOdtwarzanie lub pauza wideo/audio.
endedMedia zakończone.
volumechangeZmieniono głośność.
timeupdateAktualizacja bieżącego czasu odtwarzania.

🧱 Inne przydatne

NazwaOpis
drag / dropPrzeciąganie i upuszczanie elementów.
copy / paste / cutOperacje schowka.
visibilitychangeZakładka aktywna lub ukryta (np. zmiana kart).
